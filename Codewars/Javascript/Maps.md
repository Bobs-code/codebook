# 8kyu Map Function

## 8 kyu - Javascript - Completed: 12/13/2022

*Given an array of integers as strings and numbers, return the sum of the array values as if all values were numbers. Return your answer as a number*

```JavaScript

const arrOne = [2, 3, 4]

function maps(x){
  return x.map((i) => i * 2)
}

maps(arrOne) 
// [4, 6, 8]

```
